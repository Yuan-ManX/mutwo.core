mutwo.generators.toussaint.euclidean.euclidean(size: int, distribution: int) → Tuple[int][source]¶

Return euclidean rhythm as described in a 2005 paper by Godfried Toussaint.

Parameters

  • size – how many beats the rhythm contains

  • distribution – how many beats are played

The title of Toussaints paper is “The Euclidean Algorithm Generates Traditional Musical Rhythms”.

mutwo.generators.toussaint.reflection_rhythms module¶

Functions to build reflection rhythms.

The reflection rhythm algorithms has been described by Godfried T. Toussaint in his paper ‘Generating “Good” Musical Rhythms Algorithmically’ [2010]. Toussaint defines a reflection rhythm as any rhythm “which mirror symmetry […] about some axis is equal to its complementary rhythm”.

mutwo.generators.toussaint.reflection_rhythms.alternating_hands(seed_rhythm: Tuple[int]) → Tuple[Tuple[int]][source]¶

Generates rhythm using the alternating hands method described by G. T. Toussaint.

Parameters

seed_rhythm – rhythm that shall be distributed on two hands.

Returns

Return nested tuple that contains two tuple where each tuple represents one rhythm (both rhythms are complementary to each other). The rhythms are encoded in absolute time values.

The alternating hands algorithm has been described by Godfried T. Toussaint in his paper ‘Generating “Good” Musical Rhythms Algorithmically’.

mutwo.generators.toussaint.reflection_rhythms.paradiddle(size: int) → Tuple[Tuple[int]][source]¶

Generates rhythm using the paraddidle method described by G. T. Toussaint.

Parameters

size – how many beats the resulting rhythm shall last. ‘Size’ has to be divisible by 2 because of the symmetrical structure of the generated rhythm.

Returns

Return nested tuple that contains two tuple where each tuple represents one rhythm (both rhythms are complementary to each other). The rhythms are encoded in absolute time values.

The paradiddle algorithm has been described by Godfried T. Toussaint in his paper ‘Generating “Good” Musical Rhythms Algorithmically’.
